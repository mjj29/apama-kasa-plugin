kasa
Event Response

Response event from asynchronous Kasa methods.

Contains a request id matching the request id returned from the action and the return value of the method. Should be listened for like this: 

	monitor.subscribe(kasa.Kasa.CHANNEL);
	on kasa.Response(requestId=kasa.Kasa.discover()) as response {
	}
